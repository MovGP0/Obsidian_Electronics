### 4-Ring-Farbcode nach DIN 41429

| Kennfarbe | 1.Ziffer | 2.Ziffer | Multiplikator                   |   Toleranz |
| --------- | -------: | -------: | ------------------------------- | ---------: |
| *(keine)* |          |          |                                 | $\pm 20\%$ |
| silber    |          |          | $\times\ 10\ \mathrm{m}\Omega$  | $\pm 10\%$ |
| gold      |          |          | $\times\ 100\ \mathrm{m}\Omega$ | $\pm\ 5\%$ |
| schwarz   |          |        0 | $\times\ 1\ \Omega$             | $\pm 20\%$ |
| braun     |        1 |        1 | $\times\ 10\ \Omega$            | $\pm\ 1\%$ |
| rot       |        2 |        2 | $\times\ 100\ \Omega$           | $\pm\ 2\%$ |
| orange    |        3 |        3 | $\times\ 1\ \mathrm{k}\Omega$   |            |
| gelb      |        4 |        4 | $\times\ 10\ \mathrm{k}\Omega$  |            |
| grün      |        5 |        5 | $\times\ 100\ \mathrm{k}\Omega$ |     ±0,5 % |
| blau      |        6 |        6 | $\times\ 1\,\mathrm{M}\Omega$   |    ±0,25 % |
| violett   |        7 |        7 | $\times\ 10\,\mathrm{M}\Omega$  |     ±0,1 % |
| grau      |        8 |        8 | $\times\ 100\,\mathrm{M}\Omega$ |    ±0,05 % |
| weiß      |        9 |        9 |                                 |            |

> [!example]
> A resistor with $4.7 k\Omega$ has the ring colors yellow ($4$), violet ($7$), red ($\times\,100\,\Omega$) and silver ($\pm 10\ \%$)

### 5–6-Ring-Farbcode nach IEC 60062

| Kennfarbe | 1.Ziffer | 2.Ziffer | 3.Ziffer | Multiplikator   | Toleranz | Temperatur-koeffizient |
| :-------: | -------: | -------: | -------: | --------------- | -------- | ---------------------- |
|   pink    |          |          |          |                 |          |                        |
|  silber   |          |          |          | × 10 m$\Omega$  |          |                        |
|   gold    |          |          |          | × 100 m$\Omega$ | ± 5 %    |                        |
|  schwarz  |          |        0 |        0 | × 1 $\Omega$    |          | ±250 ppm/K             |
|   braun   |        1 |        1 |        1 | × 10 $\Omega$   | ±1 %     | ±100 ppm/K             |
|    rot    |        2 |        2 |        2 | × 100 $\Omega$  | ±2 %     | ± 50 ppm/K             |
|  orange   |        3 |        3 |        3 | × 1 k$\Omega$   |          | ± 15 ppm/K             |
|   gelb    |        4 |        4 |        4 | × 10 k$\Omega$  |          | ± 25 ppm/K             |
|   grün    |        5 |        5 |        5 | × 100 k$\Omega$ | ±0,5%    | ± 20 ppm/K             |
|   blau    |        6 |        6 |        6 | × 1 M$\Omega$   | ±0,25 %  | ± 10 ppm/K             |
|  violett  |        7 |        7 |        7 | × 10 M$\Omega$  | ±0,1%    | ± 5 ppm/K              |
|   grau    |        8 |        8 |        8 | × 100 M$\Omega$ | ±0,05 %  | ± 1 ppm/K              |
|   weiß    |        9 |        9 |        9 |                 |          |                        |

> [!example]
> A resistor with $4.75\,\mathrm{k\Omega}$ has the ring colors yellow ($4$), violet ($7$), green ($5$), brown ($×10\, \mathrm{\Omega}$), brown (±1 %), and blue (±10 ppm/K)

> [!note]
> Der Temperaturkoeffizient wird in der Regel nur angegeben, wenn er kleiner als 50 ppm/K ist.
