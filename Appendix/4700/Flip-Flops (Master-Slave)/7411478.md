The `74x11478` entry in this catalog is treated here as a metastability-resistant multi-stage D flip-flop function.

This note documents a **functionally equivalent CMOS PSpice model** for a simplified package-level device rather than any one vendor-specific TTL or advanced CMOS implementation.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7411478`
- reusable CMOS helper blocks for inversion, D latching, and positive-edge-triggered storage
- one packaged device containing 4 independent two-stage synchronizer channels

## Model Scope

- Legacy references for `11478` differ: later TI selection guides list a dual-ranked D-type device in the wider `x11478` family, while this project overview classifies `11478` as a metastable-resistant master-slave part.
- To stay aligned with the overview article, this note models `74x11478` as 4 independent positive-edge-triggered two-stage synchronizer channels with true outputs only.
- The model is a functional abstraction intended for logic-level PSpice work; it does not model analog metastability resolution time or vendor-specific hardening structures.
- The port order is logical and grouped by channel, not by physical package pins.

## Port Order

```cir
.SUBCKT CMOS_7411478
+ CLK
+ D1 Q1 D2 Q2 D3 Q3 D4 Q4
+ VDD VSS
```

## Function

For each channel:

- on each rising edge of `CLK`, data is sampled into an internal first stage
- on the next rising edge, that sampled value is transferred to `Q`

This is equivalent to a 2-stage synchronizer chain per channel.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to any original TTL or advanced CMOS family member.

**7411478.cir**
```cir
* Functional CMOS 74x11478 model.
* Each channel is represented as a two-stage positive-edge synchronizer.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DFF_P D CLK Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XM1 D CLKN MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLK Q QN VDD VSS CMOS_DLAT_H
.ENDS

.SUBCKT CMOS_7411478
+ CLK
+ D1 Q1 D2 Q2 D3 Q3 D4 Q4
+ VDD VSS

XF1A D1 CLK Q1M Q1MN VDD VSS CMOS_DFF_P
XF1B Q1M CLK Q1 Q1N VDD VSS CMOS_DFF_P
XF2A D2 CLK Q2M Q2MN VDD VSS CMOS_DFF_P
XF2B Q2M CLK Q2 Q2N VDD VSS CMOS_DFF_P
XF3A D3 CLK Q3M Q3MN VDD VSS CMOS_DFF_P
XF3B Q3M CLK Q3 Q3N VDD VSS CMOS_DFF_P
XF4A D4 CLK Q4M Q4MN VDD VSS CMOS_DFF_P
XF4B Q4M CLK Q4 Q4N VDD VSS CMOS_DFF_P
.ENDS
```

## Example Usage

```cir
.include "7411478.cir"

VDD vdd 0 DC 5
VCLK clk 0 PULSE(0 5 0 1N 1N 20N 40N)
VD1 d1 0 PULSE(0 5 0 1N 1N 70N 140N)

XU1
+ clk
+ d1 q1 0 q2 0 q3 0 q4
+ vdd 0
+ CMOS_7411478

.tran 1N 300N
.probe V(q1)
.end
```
