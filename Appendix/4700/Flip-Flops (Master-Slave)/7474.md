The `74x74` is the dual D-type positive-edge-triggered flip-flop with preset and clear in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7474`
- reusable CMOS helper blocks for inversion, level-sensitive D latching, and master-slave edge-triggered storage
- one packaged device containing 2 independent D flip-flops

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x74` function, not an electrical model of a specific TTL or CMOS vendor family.
- Each section is positive-edge triggered.
- Preset and clear are asynchronous active-low inputs and override clocked operation.
- Both true and complement outputs are provided for each section.
- The port order is logical and grouped by section.

## Port Order

```cir
.SUBCKT CMOS_7474
+ PRE1N CLR1N CLK1 D1 Q1 Q1N
+ PRE2N CLR2N CLK2 D2 Q2 Q2N
+ VDD VSS
```

## Function

For each section:

- if `CLR_N = 0`, then `Q = 0`
- else if `PRE_N = 0`, then `Q = 1`
- else on the rising edge of `CLK`, `Q = D`
- `Q_N = not(Q)`

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7474.cir**
```cir
* Functional CMOS 74x74 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12
.MODEL SWSR SW VT=2.5 VH=0.1 RON=1 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H_AR D EN SET CLR Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
SSET NSTORE VDD SET VSS SWSR
SCLR NSTORE VSS CLR VSS SWSR
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DFF_P_AR D CLK PREN CLRN Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XI2 PREN PRE VDD VSS CMOS_INV
XI3 CLRN CLR VDD VSS CMOS_INV
XM1 D CLKN MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLK PRE CLR Q QN VDD VSS CMOS_DLAT_H_AR
.ENDS

.SUBCKT CMOS_7474
+ PRE1N CLR1N CLK1 D1 Q1 Q1N
+ PRE2N CLR2N CLK2 D2 Q2 Q2N
+ VDD VSS

XF1 D1 CLK1 PRE1N CLR1N Q1 Q1N VDD VSS CMOS_DFF_P_AR
XF2 D2 CLK2 PRE2N CLR2N Q2 Q2N VDD VSS CMOS_DFF_P_AR
.ENDS
```

## Example Usage

```cir
.include "7474.cir"

VDD vdd 0 DC 5
VPRE pre1n 0 DC 1
VCLR clr1n 0 DC 1
VCLK clk1 0 PULSE(0 5 0 1N 1N 20N 40N)
VD d1 0 PULSE(0 5 0 1N 1N 60N 120N)

XU1
+ pre1n clr1n clk1 d1 q1 q1n
+ vdd vdd 0 0 q2 q2n
+ vdd 0
+ CMOS_7474

.tran 1N 200N
.probe V(q1) V(q1n)
.end
```
