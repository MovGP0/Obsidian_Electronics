The `74x114` is treated here as a dual J-K negative-edge-triggered flip-flop with preset and clear in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than any one original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74114`
- reusable CMOS helper blocks for inversion, logic composition, D latching, and J-K next-state generation
- one packaged device containing 2 independent J-K flip-flops

## Model Scope

- Legacy literature around the `74x114` family varies by vendor and successor logic family. This note follows the overview classification used in this project: negative-edge-triggered J-K sections with asynchronous active-low preset and clear.
- The model is a functional abstraction for logic simulation, not a transistor-accurate reproduction.
- True and complement outputs are provided for each section.
- The port order is logical and grouped by section.

## Port Order

```cir
.SUBCKT CMOS_74114
+ PRE1N CLR1N CLK1 J1 K1 Q1 Q1N
+ PRE2N CLR2N CLK2 J2 K2 Q2 Q2N
+ VDD VSS
```

## Function

For each section:

- if `CLR_N = 0`, then `Q = 0`
- else if `PRE_N = 0`, then `Q = 1`
- else on the falling edge of `CLK`, the next J-K state is applied

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74114.cir**
```cir
* Functional CMOS 74x114 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12
.MODEL SWSR SW VT=2.5 VH=0.1 RON=1 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NAND2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NOR2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H_AR D EN SET CLR Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
SSET NSTORE VDD SET VSS SWSR
SCLR NSTORE VSS CLR VSS SWSR
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DFF_N_AR D CLK PREN CLRN Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XI2 PREN PRE VDD VSS CMOS_INV
XI3 CLRN CLR VDD VSS CMOS_INV
XM1 D CLK MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLKN PRE CLR Q QN VDD VSS CMOS_DLAT_H_AR
.ENDS

.SUBCKT CMOS_JK_TO_D J K Q QN D VDD VSS
XI1 K KN VDD VSS CMOS_INV
XA1 J QN N1 VDD VSS CMOS_AND2
XA2 KN Q N2 VDD VSS CMOS_AND2
XO1 N1 N2 D VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_JKFF_N_AR J K CLK PREN CLRN Q QN VDD VSS
XJ1 J K Q QN D VDD VSS CMOS_JK_TO_D
XD1 D CLK PREN CLRN Q QN VDD VSS CMOS_DFF_N_AR
.ENDS

.SUBCKT CMOS_74114
+ PRE1N CLR1N CLK1 J1 K1 Q1 Q1N
+ PRE2N CLR2N CLK2 J2 K2 Q2 Q2N
+ VDD VSS

XF1 J1 K1 CLK1 PRE1N CLR1N Q1 Q1N VDD VSS CMOS_JKFF_N_AR
XF2 J2 K2 CLK2 PRE2N CLR2N Q2 Q2N VDD VSS CMOS_JKFF_N_AR
.ENDS
```

## Example Usage

```cir
.include "74114.cir"

VDD vdd 0 DC 5
VPRE pre1n 0 DC 1
VCLR clr1n 0 DC 1
VCLK clk1 0 PULSE(0 5 0 1N 1N 20N 40N)
VJ j1 0 DC 1
VK k1 0 DC 0

XU1
+ pre1n clr1n clk1 j1 k1 q1 q1n
+ vdd vdd 0 0 0 q2 q2n
+ vdd 0
+ CMOS_74114

.tran 1N 200N
.probe V(q1)
.end
```
