The `74x171` is the quad D-type flip-flop with clear in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74171`
- reusable CMOS helper blocks for inversion, level-sensitive D latching, and positive-edge-triggered storage
- one packaged device containing 4 D flip-flops

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x171` function, not an electrical model of a specific TTL or CMOS vendor family.
- The device is modeled as a quad positive-edge-triggered D register with a common asynchronous active-low clear.
- Both true and complement outputs are exposed in this simplified model because that is the most useful abstract form for simulation, even though package details vary across legacy references.
- The port order is logical and grouped by bit rather than by physical pin numbering.

## Port Order

```cir
.SUBCKT CMOS_74171
+ CLRN CLK
+ D1 Q1 Q1N D2 Q2 Q2N
+ D3 Q3 Q3N D4 Q4 Q4N
+ VDD VSS
```

## Function

For each bit:

- if `CLR_N = 0`, then `Q = 0`
- else on the rising edge of `CLK`, `Q = D`
- `Q_N = not(Q)`

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74171.cir**
```cir
* Functional CMOS 74x171 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12
.MODEL SWSR SW VT=2.5 VH=0.1 RON=1 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H_AR D EN SET CLR Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
SSET NSTORE VDD SET VSS SWSR
SCLR NSTORE VSS CLR VSS SWSR
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DFF_P_CLR D CLK CLRN Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XM1 D CLKN MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLK VSS CLR Q QN VDD VSS CMOS_DLAT_H_AR
XI2 CLRN CLR VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_74171
+ CLRN CLK
+ D1 Q1 Q1N D2 Q2 Q2N
+ D3 Q3 Q3N D4 Q4 Q4N
+ VDD VSS

XF1 D1 CLK CLRN Q1 Q1N VDD VSS CMOS_DFF_P_CLR
XF2 D2 CLK CLRN Q2 Q2N VDD VSS CMOS_DFF_P_CLR
XF3 D3 CLK CLRN Q3 Q3N VDD VSS CMOS_DFF_P_CLR
XF4 D4 CLK CLRN Q4 Q4N VDD VSS CMOS_DFF_P_CLR
.ENDS
```

## Example Usage

```cir
.include "74171.cir"

VDD vdd 0 DC 5
VCLR clrn 0 DC 1
VCLK clk 0 PULSE(0 5 0 1N 1N 20N 40N)
VD1 d1 0 PULSE(0 5 0 1N 1N 60N 120N)

XU1
+ clrn clk
+ d1 q1 q1n 0 q2 q2n
+ 0 q3 q3n 0 q4 q4n
+ vdd 0
+ CMOS_74171

.tran 1N 200N
.probe V(q1) V(q1n)
.end
```
