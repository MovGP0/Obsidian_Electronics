The `74x174` is the hex D-type flip-flop with clear in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74174`
- reusable CMOS helper blocks for inversion, D latching, and positive-edge-triggered storage
- one packaged device containing 6 D flip-flops

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x174` function, not an electrical model of a specific TTL or CMOS vendor family.
- All 6 sections share a common clock and a common asynchronous active-low clear.
- True outputs are modeled; complement outputs are not exposed for this device.
- The port order is logical and grouped by bit.

## Port Order

```cir
.SUBCKT CMOS_74174
+ CLRN CLK
+ D1 Q1 D2 Q2 D3 Q3
+ D4 Q4 D5 Q5 D6 Q6
+ VDD VSS
```

## Function

For each bit:

- if `CLR_N = 0`, then `Q = 0`
- else on the rising edge of `CLK`, `Q = D`

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74174.cir**
```cir
* Functional CMOS 74x174 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12
.MODEL SWSR SW VT=2.5 VH=0.1 RON=1 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H_AR D EN SET CLR Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
SSET NSTORE VDD SET VSS SWSR
SCLR NSTORE VSS CLR VSS SWSR
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DFF_P_CLR D CLK CLRN Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XI2 CLRN CLR VDD VSS CMOS_INV
XM1 D CLKN MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLK VSS CLR Q QN VDD VSS CMOS_DLAT_H_AR
.ENDS

.SUBCKT CMOS_74174
+ CLRN CLK
+ D1 Q1 D2 Q2 D3 Q3
+ D4 Q4 D5 Q5 D6 Q6
+ VDD VSS

XF1 D1 CLK CLRN Q1 Q1N VDD VSS CMOS_DFF_P_CLR
XF2 D2 CLK CLRN Q2 Q2N VDD VSS CMOS_DFF_P_CLR
XF3 D3 CLK CLRN Q3 Q3N VDD VSS CMOS_DFF_P_CLR
XF4 D4 CLK CLRN Q4 Q4N VDD VSS CMOS_DFF_P_CLR
XF5 D5 CLK CLRN Q5 Q5N VDD VSS CMOS_DFF_P_CLR
XF6 D6 CLK CLRN Q6 Q6N VDD VSS CMOS_DFF_P_CLR
.ENDS
```

## Example Usage

```cir
.include "74174.cir"

VDD vdd 0 DC 5
VCLR clrn 0 DC 1
VCLK clk 0 PULSE(0 5 0 1N 1N 20N 40N)
VD1 d1 0 PULSE(0 5 0 1N 1N 60N 120N)

XU1
+ clrn clk
+ d1 q1 0 q2 0 q3
+ 0 q4 0 q5 0 q6
+ vdd 0
+ CMOS_74174

.tran 1N 200N
.probe V(q1)
.end
```
