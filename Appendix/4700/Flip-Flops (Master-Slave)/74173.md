The `74x173` is the quad D-type flip-flop with clear, gated data entry, and 3-state outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74173`
- reusable CMOS helper blocks for inversion, basic logic, D flip-flops, multiplexing, and tri-state outputs
- one packaged device containing 4 registered storage bits with common control inputs

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x173` function, not an electrical model of a specific TTL or CMOS vendor family.
- Data is loaded on the rising edge of the common clock only when both active-low data-enable inputs are LOW.
- Outputs are enabled only when both active-low output-control inputs are LOW; otherwise they are high impedance.
- Clear is asynchronous and active low.
- Internal storage remains active even when outputs are disabled.
- The port order is logical and grouped by controls and data bits.

## Port Order

```cir
.SUBCKT CMOS_74173
+ M_N N_N G1_N G2_N CLR_N CLK
+ D1 Y1 D2 Y2 D3 Y3 D4 Y4
+ VDD VSS
```

## Function

- if `CLR_N = 0`, then all stored bits reset to `0`
- else if `G1_N = 0` and `G2_N = 0`, the inputs are loaded on the rising edge of `CLK`
- else the register holds its previous state
- if `M_N = 0` and `N_N = 0`, outputs drive the stored values
- otherwise outputs are high impedance

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74173.cir**
```cir
* Functional CMOS 74x173 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12
.MODEL SWSR SW VT=2.5 VH=0.1 RON=1 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NAND2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NOR2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_MUX2 A B S Y VDD VSS
XI1 S SN VDD VSS CMOS_INV
XA1 A SN N1 VDD VSS CMOS_AND2
XA2 B S N2 VDD VSS CMOS_AND2
XO1 N1 N2 Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H_AR D EN SET CLR Q QN VDD VSS

SSET NSTORE VDD SET VSS SWSR
SCLR NSTORE VSS CLR VSS SWSR
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV

.ENDS

.SUBCKT CMOS_DFF_P_CLR D CLK CLRN Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XI2 CLRN CLR VDD VSS CMOS_INV
XM1 D CLKN MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLK VSS CLR Q QN VDD VSS CMOS_DLAT_H_AR
.ENDS

.SUBCKT CMOS_TS_BUF_H A EN Y VDD VSS
EIN DIN VSS VALUE = { V(A,VSS) }
S1 Y DIN EN VSS SWEN
.ENDS

.SUBCKT CMOS_74173
+ M_N N_N G1_N G2_N CLR_N CLK
+ D1 Y1 D2 Y2 D3 Y3 D4 Y4
+ VDD VSS

XN1 G1_N G2_N LOAD VDD VSS CMOS_NOR2
XN2 M_N N_N OEN VDD VSS CMOS_NOR2

XM1 Q1I D1 LOAD D1SEL VDD VSS CMOS_MUX2
XM2 Q2I D2 LOAD D2SEL VDD VSS CMOS_MUX2
XM3 Q3I D3 LOAD D3SEL VDD VSS CMOS_MUX2
XM4 Q4I D4 LOAD D4SEL VDD VSS CMOS_MUX2

XF1 D1SEL CLK CLR_N Q1I Q1N VDD VSS CMOS_DFF_P_CLR
XF2 D2SEL CLK CLR_N Q2I Q2N VDD VSS CMOS_DFF_P_CLR
XF3 D3SEL CLK CLR_N Q3I Q3N VDD VSS CMOS_DFF_P_CLR
XF4 D4SEL CLK CLR_N Q4I Q4N VDD VSS CMOS_DFF_P_CLR

XT1 Q1I OEN Y1 VDD VSS CMOS_TS_BUF_H
XT2 Q2I OEN Y2 VDD VSS CMOS_TS_BUF_H
XT3 Q3I OEN Y3 VDD VSS CMOS_TS_BUF_H
XT4 Q4I OEN Y4 VDD VSS CMOS_TS_BUF_H
.ENDS
```

## Example Usage

```cir
.include "74173.cir"

VDD vdd 0 DC 5
VM mn 0 DC 0
VN nn 0 DC 0
VG1 g1n 0 DC 0
VG2 g2n 0 DC 0
VCLR clrn 0 DC 1
VCLK clk 0 PULSE(0 5 0 1N 1N 20N 40N)
VD1 d1 0 PULSE(0 5 0 1N 1N 60N 120N)
RLOAD y1 0 100K

XU1
+ mn nn g1n g2n clrn clk
+ d1 y1 0 y2 0 y3 0 y4
+ vdd 0
+ CMOS_74173

.tran 1N 200N
.probe V(y1)
.end
```
