The `74x183` is the dual carry-save full adder in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74183`
- reusable CMOS helper blocks for logic composition and full-adder behavior
- one packaged device containing 2 independent carry-save full-adder sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x183` function, not an electrical model of a specific TTL or CMOS vendor family.
- Each section adds 3 one-bit operands and produces a sum bit plus a carry-save bit.
- The logical port order is grouped by adder section; it does not attempt to reproduce physical DIP pin numbering.
- This simplified model treats the device as 2 independent 1-bit 3:2 compressor cells, which matches the usual carry-save usage in multipliers and reduction trees.

## Port Order

```cir
.SUBCKT CMOS_74183
+ A1 B1 C1 S1 CS1
+ A2 B2 C2 S2 CS2
+ VDD VSS
```

## Function

For each section:

- `S = A xor B xor C`
- `CS = majority(A, B, C)`

where `CS` is the carry-save output.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74183.cir**
```cir
* Functional CMOS 74x183 model.

.SUBCKT CMOS_74183
+ A1 B1 C1 S1 CS1
+ A2 B2 C2 S2 CS2
+ VDD VSS

ES1 S1 VSS VALUE = { IF((V(A1,VSS)>2.5)^(V(B1,VSS)>2.5)^(V(C1,VSS)>2.5), V(VDD,VSS), 0) }
ECS1 CS1 VSS VALUE = {
+ IF(
+   ((V(A1,VSS)>2.5)&(V(B1,VSS)>2.5))|
+   ((V(A1,VSS)>2.5)&(V(C1,VSS)>2.5))|
+   ((V(B1,VSS)>2.5)&(V(C1,VSS)>2.5)),
+   V(VDD,VSS),
+   0)
+ }

ES2 S2 VSS VALUE = { IF((V(A2,VSS)>2.5)^(V(B2,VSS)>2.5)^(V(C2,VSS)>2.5), V(VDD,VSS), 0) }
ECS2 CS2 VSS VALUE = {
+ IF(
+   ((V(A2,VSS)>2.5)&(V(B2,VSS)>2.5))|
+   ((V(A2,VSS)>2.5)&(V(C2,VSS)>2.5))|
+   ((V(B2,VSS)>2.5)&(V(C2,VSS)>2.5)),
+   V(VDD,VSS),
+   0)
+ }
.ENDS
```

## Example Usage

```cir
.include "74183.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1
VB1 b1 0 DC 1
VC1 c1 0 DC 0

XU1
+ a1 b1 c1 s1 cs1
+ 0 1 1 s2 cs2
+ vdd 0
+ CMOS_74183

.op
.probe V(s1) V(cs1) V(s2) V(cs2)
.end
```
