The `74x583` is the 4-bit BCD adder in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74583`
- behavioral BCD addition logic for one decimal digit plus carry input
- one packaged device with BCD sum outputs and decimal carry output

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x583` function, not an electrical model of a specific TTL or CMOS vendor family.
- The device is intended for valid BCD digits `0` through `9` on both operand inputs.
- The model produces a decimal carry when the sum exceeds 9 and returns the ones digit in BCD on `S0..S3`.
- When used with one operand tied low, the device also acts as a simple binary-to-BCD converter for inputs up to 15, matching the common FAST-family description.
- The port order is logical and grouped by operands, carry input, sum outputs, and carry output.

## Port Order

```cir
.SUBCKT CMOS_74583
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ CIN
+ S0 S1 S2 S3
+ COUT
+ VDD VSS
```

## Function

For valid BCD digits `A` and `B`:

- compute `T = A + B + CIN`
- if `T < 10`, then `COUT = 0` and `S = T`
- if `T >= 10`, then `COUT = 1` and `S = T - 10`

where `S` is encoded in BCD on `S3..S0`.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74583.cir**
```cir
* Functional CMOS 74x583 model.

.SUBCKT CMOS_74583
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ CIN
+ S0 S1 S2 S3
+ COUT
+ VDD VSS

EADEC ADEC VSS VALUE = { IF(V(A0,VSS)>2.5,1,0) + IF(V(A1,VSS)>2.5,2,0) + IF(V(A2,VSS)>2.5,4,0) + IF(V(A3,VSS)>2.5,8,0) }
EBDEC BDEC VSS VALUE = { IF(V(B0,VSS)>2.5,1,0) + IF(V(B1,VSS)>2.5,2,0) + IF(V(B2,VSS)>2.5,4,0) + IF(V(B3,VSS)>2.5,8,0) }
ETOT TOT VSS VALUE = { V(ADEC,VSS) + V(BDEC,VSS) + IF(V(CIN,VSS)>2.5,1,0) }
ECOUT COUT VSS VALUE = { IF(V(TOT,VSS)>9.5, V(VDD,VSS), 0) }
ESUM DEC VSS VALUE = { IF(V(TOT,VSS)>9.5, V(TOT,VSS)-10, V(TOT,VSS)) }
ES0 S0 VSS VALUE = { IF(MOD(V(DEC,VSS),2)>0.5, V(VDD,VSS), 0) }
ES1 S1 VSS VALUE = { IF(MOD(INT(V(DEC,VSS)/2),2)>0.5, V(VDD,VSS), 0) }
ES2 S2 VSS VALUE = { IF(MOD(INT(V(DEC,VSS)/4),2)>0.5, V(VDD,VSS), 0) }
ES3 S3 VSS VALUE = { IF(MOD(INT(V(DEC,VSS)/8),2)>0.5, V(VDD,VSS), 0) }
.ENDS
```

## Example Usage

```cir
.include "74583.cir"

VDD vdd 0 DC 5
VA0 a0 0 DC 1
VA1 a1 0 DC 0
VA2 a2 0 DC 0
VA3 a3 0 DC 0
VB0 b0 0 DC 1
VB1 b1 0 DC 1
VB2 b2 0 DC 0
VB3 b3 0 DC 0
VCIN cin 0 DC 0

XU1
+ a0 a1 a2 a3
+ b0 b1 b2 b3
+ cin
+ s0 s1 s2 s3
+ cout
+ vdd 0
+ CMOS_74583

.op
.probe V(s0) V(s1) V(s2) V(s3) V(cout)
.end
```
