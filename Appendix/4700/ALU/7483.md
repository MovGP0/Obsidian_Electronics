The `74x83` is the 4-bit binary full adder in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7483`
- reusable CMOS helper blocks for inversion, XOR, and full-adder construction
- one packaged device containing a 4-bit binary adder with carry input and carry output

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x83` function, not an electrical model of a specific TTL or CMOS vendor family.
- The adder operates on active-high true-form operands and carry signals.
- The port order is logical and grouped by operand words, carry input, sum outputs, and carry output.
- This simplified model preserves arithmetic behavior only; it does not attempt to reproduce TTL timing skew or transistor-level carry structure.

## Port Order

```cir
.SUBCKT CMOS_7483
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ CIN
+ S0 S1 S2 S3
+ COUT
+ VDD VSS
```

## Function

The device performs:

`{COUT, S3..S0} = A + B + CIN`

where `A` and `B` are 4-bit unsigned binary words.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7483.cir**
```cir
* Functional CMOS 74x83 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NAND2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NOR2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_XOR2 A B Y VDD VSS
E1 Y VSS VALUE = { IF((V(A,VSS)>2.5)^(V(B,VSS)>2.5), V(VDD,VSS), 0) }
.ENDS

.SUBCKT CMOS_FULLADD A B CIN S COUT VDD VSS
XX1 A B X1 VDD VSS CMOS_XOR2
XX2 X1 CIN S VDD VSS CMOS_XOR2
XA1 A B C1 VDD VSS CMOS_AND2
XA2 A CIN C2 VDD VSS CMOS_AND2
XA3 B CIN C3 VDD VSS CMOS_AND2
XO1 C1 C2 C4 VDD VSS CMOS_OR2
XO2 C4 C3 COUT VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_7483
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ CIN
+ S0 S1 S2 S3
+ COUT
+ VDD VSS

XF0 A0 B0 CIN S0 C1 VDD VSS CMOS_FULLADD
XF1 A1 B1 C1 S1 C2 VDD VSS CMOS_FULLADD
XF2 A2 B2 C2 S2 C3 VDD VSS CMOS_FULLADD
XF3 A3 B3 C3 S3 COUT VDD VSS CMOS_FULLADD
.ENDS
```

## Example Usage

```cir
.include "7483.cir"

VDD vdd 0 DC 5
VA0 a0 0 DC 1
VA1 a1 0 DC 0
VA2 a2 0 DC 1
VA3 a3 0 DC 0
VB0 b0 0 DC 1
VB1 b1 0 DC 1
VB2 b2 0 DC 0
VB3 b3 0 DC 0
VCIN cin 0 DC 0

XU1
+ a0 a1 a2 a3
+ b0 b1 b2 b3
+ cin
+ s0 s1 s2 s3
+ cout
+ vdd 0
+ CMOS_7483

.op
.probe V(s0) V(s1) V(s2) V(s3) V(cout)
.end
```
