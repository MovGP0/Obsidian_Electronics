The `74x881` is a 4-bit arithmetic logic unit / function generator variant in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL or FAST transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74881`
- a wrapper around the already documented `CMOS_74181` ALU core
- one additional status output `ST` that flags a zero result in this simplified model

## Model Scope

- Legacy FAST-family references describe `74x881` as an **ALU/function generator**, while catalogs in this project summarize it as a 4-bit ALU with status check.
- To stay consistent with the already finished `74181` article and avoid duplicating a second full ALU core, this note models `74x881` as a **74181-compatible 4-bit ALU core** plus a derived status output.
- The wrapper expects the `CMOS_74181` subcircuit from the existing [`74181.md`](D:/GitHub/Obsidian_Science/Electronics/Appendix/4700/ALU/74181.md) note to be available in the simulation.
- In this simplified abstraction, `ST = 1` when the 4-bit result word is zero and `ST = 0` otherwise.
- The original vendor-specific status conventions are not reproduced electrically; the note documents the abstraction explicitly instead.

## Port Order

```cir
.SUBCKT CMOS_74881
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ S0 S1 S2 S3
+ M CIN
+ F0 F1 F2 F3
+ COUT P G ST
+ VDD VSS
```

## Function

- The `A`, `B`, `S`, `M`, and `CIN` inputs drive the same ALU behavior as the simplified `CMOS_74181` model.
- `F0..F3`, `COUT`, `P`, and `G` are passed through from the ALU core.
- `ST` is asserted HIGH when `F3..F0 = 0000`.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL or FAST family member.

**74881.cir**
```cir
* Functional CMOS 74x881 wrapper model.
* This model depends on the CMOS_74181 subcircuit from 74181.cir.

.SUBCKT CMOS_74881
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ S0 S1 S2 S3
+ M CIN
+ F0 F1 F2 F3
+ COUT P G ST
+ VDD VSS

XALU
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ S0 S1 S2 S3
+ M CIN
+ F0 F1 F2 F3
+ COUT P G EQB_UNUSED
+ VDD VSS
+ CMOS_74181

EST ST VSS VALUE = {
+ IF(
+   (V(F0,VSS)<0.5)&(V(F1,VSS)<0.5)&(V(F2,VSS)<0.5)&(V(F3,VSS)<0.5),
+   V(VDD,VSS),
+   0)
+ }
.ENDS
```

## Example Usage

```cir
.include "74181.cir"
.include "74881.cir"

VDD vdd 0 DC 5
VA0 a0 0 DC 1
VA1 a1 0 DC 0
VA2 a2 0 DC 1
VA3 a3 0 DC 0
VB0 b0 0 DC 1
VB1 b1 0 DC 0
VB2 b2 0 DC 1
VB3 b3 0 DC 0
VS0 s0 0 DC 1
VS1 s1 0 DC 1
VS2 s2 0 DC 1
VS3 s3 0 DC 1
VM m 0 DC 1
VCIN cin 0 DC 0

XU1
+ a0 a1 a2 a3
+ b0 b1 b2 b3
+ s0 s1 s2 s3
+ m cin
+ f0 f1 f2 f3
+ cout p g st
+ vdd 0
+ CMOS_74881

.op
.probe V(f0) V(f1) V(f2) V(f3) V(st)
.end
```
