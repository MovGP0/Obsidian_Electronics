The `74181` is a classic 4-bit *arithmetic logic unit* (ALU). 

The original IC is a TTL part with an internal gate-level structure and active-low naming in many older datasheets.

## What Is Included

- reusable CMOS gate `.SUBCKT` blocks for `INV`, `NAND2`, `NOR2`, `AND2`, `OR2`, `XOR2`, `XNOR2`
- structural helper blocks for `MUX2`, `MUX4`, `MUX16`, `FULLADD`, and `ADD4`
- a top-level `CMOS_74181` `.SUBCKT`

## Model Scope

- Inputs and outputs are **active-high**
- `M = 1` selects logic mode
- `M = 0` selects arithmetic mode
- `CIN = 1` adds one to the selected arithmetic operation
- `P` and `G` are carry propagate / carry generate outputs for arithmetic mode and are forced low in logic mode
- `AEQB` is decoded from `F0..F3` as in the original part

## Port Order

```cir
.SUBCKT CMOS_74181
+ A0 A1 A2 A3
+ B0 B1 B2 B3
+ S0 S1 S2 S3
+ M CIN
+ F0 F1 F2 F3
+ COUT P G AEQB
+ VDD VSS
```

## Logic Functions

For `M = 1`, the select lines `S3..S0` choose the 16 Boolean functions of `A` and `B`. The CMOS model uses the active-high form from the classic `74181` function table:

| S3 S2 S1 S0 | Function |
| ----------- | -------- |
| `0000` | `NOT A` |
| `0001` | `NOR(A, B)` |
| `0010` | `(NOT A) AND B` |
| `0011` | `0` |
| `0100` | `NAND(A, B)` |
| `0101` | `NOT B` |
| `0110` | `A XOR B` |
| `0111` | `A AND (NOT B)` |
| `1000` | `(NOT A) OR B` |
| `1001` | `A XNOR B` |
| `1010` | `B` |
| `1011` | `A AND B` |
| `1100` | `1` |
| `1101` | `A OR (NOT B)` |
| `1110` | `A OR B` |
| `1111` | `A` |

## Arithmetic Functions

For `M = 0`, the same select lines choose the 16 arithmetic functions.

The model follows the active-high Fairchild / TI function tables.

The operations below are the **no-carry** forms; `CIN = 1` adds one to the selected operation.

The mnemonic column is intentionally only an **approximate assembler-style gloss**. Where there is no close one-word instruction in a typical ISA, the table uses a descriptive pseudo-mnemonic instead.

| S3 S2 S1 S0 | Arithmetic Function             | Approx. Verb (`CIN = 0`) | Approx. Verb (`CIN = 1`) |
| ----------- | ------------------------------- | ------------------------ | ------------------------ |
| `0000`      | $A$                             | `MOV A` / `PASS A`       | `INC A`                  |
| `0001`      | $A + B$                         | `ADD A, B`               | `ADC A, B`               |
| `0010`      | $A + \neg B$                    | `SUB A, B` minus 1       | `SUB A, B`               |
| `0011`      | $-1$                            | `SET -1`                 | `CLR`                    |
| `0100`      | $A + A \land (\neg B)$          | `ADD A, (A AND NOT B)`   | `INC` after masked add   |
| `0101`      | $(A \lor B) + A \land (\neg B)$ | `ADD merge(A, B)`        | `INC` after merged add   |
| `0110`      | $A - B - 1$                     | `SBB A, B`               | `SUB A, B`               |
| `0111`      | $(A \land B) - 1$               | `DEC (A AND B)`          | `AND A, B`               |
| `1000`      | $A + A \land B$                 | `ADD A, (A AND B)`       | `INC` after masked add   |
| `1001`      | $A + B$                         | `ADD A, B`               | `ADC A, B`               |
| `1010`      | $(A \lor \neg B) + A \land B$   | `ADD mixed(A, B)`        | `INC` after mixed add    |
| `1011`      | $(A \land B) - 1$               | `DEC (A AND B)`          | `AND A, B`               |
| `1100`      | $A + A$                         | `SHL A` / `ADD A, A`     | `RCL A` / `ADD A, A + 1` |
| `1101`      | $(A \lor B) + A$                | `ADD A, (A OR B)`        | `INC` after OR-add       |
| `1110`      | $(A \lor \neg B) + A$           | `ADD A, (A OR NOT B)`    | `INC` after ORN-add      |
| `1111`      | $A - 1$                         | `DEC A`                  | `MOV A` / `PASS A`       |

> [!note]
> For simulation work it is often more convenient to use an **active-high, functionally equivalent CMOS model** instead of trying to reproduce the original bipolar transistor network.

> [!note]

**Example usage**
```cir
.include "74181-gate-level.cir"

VDD vdd 0 DC 5

VA0 a0 0 DC 1
VA1 a1 0 DC 0
VA2 a2 0 DC 1
VA3 a3 0 DC 0

VB0 b0 0 DC 1
VB1 b1 0 DC 1
VB2 b2 0 DC 0
VB3 b3 0 DC 0

VS0 s0 0 DC 1
VS1 s1 0 DC 0
VS2 s2 0 DC 0
VS3 s3 0 DC 0

VM   m   0 DC 0
VCIN cin 0 DC 0

XALU
+ a0 a1 a2 a3
+ b0 b1 b2 b3
+ s0 s1 s2 s3
+ m cin
+ f0 f1 f2 f3
+ cout p g aeqb
+ vdd 0
+ CMOS_74181

.op
.probe V(f0) V(f1) V(f2) V(f3) V(cout) V(p) V(g) V(aeqb)
.end
```
