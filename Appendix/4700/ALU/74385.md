The `74x385` is the quad serial adder/subtractor in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74385`
- reusable CMOS helper blocks for inversion, XOR, logic composition, and edge-triggered storage
- one packaged device containing 4 independent synchronous serial adder/subtractor channels with common clock and reset

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x385` function, not an electrical model of a specific TTL or CMOS vendor family.
- Each channel contains an internal sum register and an internal carry/borrow state register updated on the rising clock edge.
- `S = 0` selects add mode and `S = 1` selects subtract mode using 2's-complement serial arithmetic.
- A LOW `MR_N` asynchronously clears the sum register and initializes the internal carry state:
  - to `0` in add mode
  - to `1` in subtract mode
- The port order is logical and grouped by common controls and per-channel data.

## Port Order

```cir
.SUBCKT CMOS_74385
+ CP MR_N
+ A0 B0 S0 F0
+ A1 B1 S1 F1
+ A2 B2 S2 F2
+ A3 B3 S3 F3
+ VDD VSS
```

## Function

For each channel, on the rising edge of `CP`:

- in add mode, the device accumulates `A + B + carry_state`
- in subtract mode, the device accumulates `A + not(B) + carry_state`

The registered sum appears at `F`. The carry state is internal and advances from bit to bit over time in serial operation.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74385.cir**
```cir
* Functional CMOS 74x385 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12
.MODEL SWSR SW VT=2.5 VH=0.1 RON=1 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NAND2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NOR2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_XOR2 A B Y VDD VSS
E1 Y VSS VALUE = { IF((V(A,VSS)>2.5)^(V(B,VSS)>2.5), V(VDD,VSS), 0) }
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DLAT_H_AR D EN SET CLR Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
SSET NSTORE VDD SET VSS SWSR
SCLR NSTORE VSS CLR VSS SWSR
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_DFF_P_AR D CLK PREN CLRN Q QN VDD VSS
XI1 CLK CLKN VDD VSS CMOS_INV
XI2 PREN PRE VDD VSS CMOS_INV
XI3 CLRN CLR VDD VSS CMOS_INV
XM1 D CLKN MQ MQN VDD VSS CMOS_DLAT_H
XS1 MQ CLK PRE CLR Q QN VDD VSS CMOS_DLAT_H_AR
.ENDS

.SUBCKT CMOS_SERIAL_ADD_SUB A B SEL CIN SUM COUT VDD VSS
XX1 B SEL BX VDD VSS CMOS_XOR2
XX2 A BX X1 VDD VSS CMOS_XOR2
XX3 X1 CIN SUM VDD VSS CMOS_XOR2
XA1 A BX C1 VDD VSS CMOS_AND2
XA2 A CIN C2 VDD VSS CMOS_AND2
XA3 BX CIN C3 VDD VSS CMOS_AND2
XO1 C1 C2 C4 VDD VSS CMOS_OR2
XO2 C4 C3 COUT VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_74385_BIT CP MRN A B SEL F VDD VSS
XI0 SEL SELN VDD VSS CMOS_INV
XA0 MRN SELN CLR_ADD VDD VSS CMOS_AND2
XA1 MRN SEL PRE_SUB VDD VSS CMOS_AND2
XI1 CLR_ADD CLRN VDD VSS CMOS_INV
XI2 PRE_SUB PREN VDD VSS CMOS_INV

XU1 A B SEL CSTATE SUMNEXT CNEXT VDD VSS CMOS_SERIAL_ADD_SUB
XF1 SUMNEXT CP PREN CLRN F FQN VDD VSS CMOS_DFF_P_AR
XF2 CNEXT CP PREN CLRN CSTATE CSTATEN VDD VSS CMOS_DFF_P_AR
.ENDS

.SUBCKT CMOS_74385
+ CP MR_N
+ A0 B0 S0 F0
+ A1 B1 S1 F1
+ A2 B2 S2 F2
+ A3 B3 S3 F3
+ VDD VSS

XB0 CP MR_N A0 B0 S0 F0 VDD VSS CMOS_74385_BIT
XB1 CP MR_N A1 B1 S1 F1 VDD VSS CMOS_74385_BIT
XB2 CP MR_N A2 B2 S2 F2 VDD VSS CMOS_74385_BIT
XB3 CP MR_N A3 B3 S3 F3 VDD VSS CMOS_74385_BIT
.ENDS
```

## Example Usage

```cir
.include "74385.cir"

VDD vdd 0 DC 5
VCP cp 0 PULSE(0 5 0 1N 1N 20N 40N)
VMR mrn 0 PULSE(0 5 0 1N 1N 8N 200N)
VA0 a0 0 PULSE(0 5 0 1N 1N 60N 120N)
VB0 b0 0 DC 1
VS0 s0 0 DC 0

XU1
+ cp mrn
+ a0 b0 s0 f0
+ 0 0 0 f1
+ 0 0 0 f2
+ 0 0 0 f3
+ vdd 0
+ CMOS_74385

.tran 1N 200N
.probe V(f0)
.end
```
