The `74x30` is the single 8-input NAND device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7430`
- reusable CMOS helper blocks for inverter, buffering, push-pull NAND stages, and open-drain outputs where needed
- one packaged NAND function with 8 inputs

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x30` function, not an electrical model of a specific TTL or CMOS vendor family.
- The port order is logical and grouped by gate; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.

## Port Order

```cir
.SUBCKT CMOS_7430
+ A B C D E F
+ G H Y VDD VSS
```

## Function

$$Y = \neg(A \land B \land C \land D \land E \land F \land G \land H)$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7430.cir**
```cir
* Functional CMOS 74x30 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_BUF A Y VDD VSS
XU1 A N1 VDD VSS CMOS_INV
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND3 A B C Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MP3 Y C VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B N2 VSS CMOSN L=1U W=1U
MN3 N2 C VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND4 A B C D Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MP3 Y C VDD VDD CMOSP L=1U W=2U
MP4 Y D VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B N2 VSS CMOSN L=1U W=1U
MN3 N2 C N3 VSS CMOSN L=1U W=1U
MN4 N3 D VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NAND2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_AND3 A B C Y VDD VSS
XA1 A B N1 VDD VSS CMOS_AND2
XA2 N1 C Y VDD VSS CMOS_AND2
.ENDS

.SUBCKT CMOS_AND4 A B C D Y VDD VSS
XA1 A B N1 VDD VSS CMOS_AND2
XA2 C D N2 VDD VSS CMOS_AND2
XA3 N1 N2 Y VDD VSS CMOS_AND2
.ENDS

.SUBCKT CMOS_NAND8 A1 A2 A3 A4 A5 A6 A7 A8 Y VDD VSS
XA1 A1 A2 A3 A4 N1 VDD VSS CMOS_AND4
XA2 A5 A6 A7 A8 N2 VDD VSS CMOS_AND4
XA3 N1 N2 N3 VDD VSS CMOS_AND2
XI1 N3 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_NAND13
+ A1 A2 A3 A4 A5 A6 A7 A8 A9 A10 A11 A12 A13
+ Y VDD VSS
XA1 A1 A2 A3 A4 N1 VDD VSS CMOS_AND4
XA2 A5 A6 A7 A8 N2 VDD VSS CMOS_AND4
XA3 A9 A10 A11 A12 N3 VDD VSS CMOS_AND4
XA4 N1 N2 N3 A13 N4 VDD VSS CMOS_AND4
XI1 N4 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OD_OUT CTRL Y VSS
MN1 Y CTRL VSS VSS CMOSN L=1U W=2U
.ENDS

.SUBCKT CMOS_7430
+ A B C D E F
+ G H Y VDD VSS

XG1 A B C D E F G H N1 VDD VSS CMOS_NAND8
EYY Y VSS VALUE = { V(N1,VSS) }
.ENDS
```

## Example Usage

```cir
.include "7430.cir"

VDD vdd 0 DC 5
VA a 0 DC 1
VB b 0 DC 1
VC c 0 DC 1
VD d 0 DC 1
VE e 0 DC 1
VF f 0 DC 1
VG g 0 DC 1
VH h 0 DC 0

XU1
+ a b c d e f g h y vdd 0
+ CMOS_7430

.op
.probe V(y)
.end
```
