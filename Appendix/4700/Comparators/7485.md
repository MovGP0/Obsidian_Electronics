The `74x85` is the 4-bit magnitude comparator in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7485`
- reusable behavioral helper blocks for 4-bit magnitude comparison
- one packaged device with cascade inputs and outputs

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x85` function, not an electrical model of a specific TTL or CMOS vendor family.
- The 3 cascade inputs are preserved so multiple devices can be chained for wider comparisons.
- In standalone use, tie `IAGTB = 0`, `IAEQB = 1`, and `IALTB = 0`.
- The port order is logical and grouped by operand bits and cascade signals.

## Port Order

```cir
.SUBCKT CMOS_7485
+ A3 A2 A1 A0 B3 B2 B1 B0
+ IAGTB IAEQB IALTB
+ OAGTB OAEQB OALTB
+ VDD VSS
```

## Function

- if `A > B`, then `OA>B = 1`, `OA=B = 0`, `OA<B = 0`
- if `A < B`, then `OA>B = 0`, `OA=B = 0`, `OA<B = 1`
- if `A = B`, then the cascade inputs determine the outputs:
  - `OA>B = IA>B`
  - `OA=B = IA=B`
  - `OA<B = IA<B`

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7485.cir**
```cir
* Functional CMOS 74x85 model.

.SUBCKT CMOS_7485
+ A3 A2 A1 A0 B3 B2 B1 B0
+ IAGTB IAEQB IALTB
+ OAGTB OAEQB OALTB
+ VDD VSS

EADEC ADEC VSS VALUE = { IF(V(A0,VSS)>2.5,1,0) + IF(V(A1,VSS)>2.5,2,0) + IF(V(A2,VSS)>2.5,4,0) + IF(V(A3,VSS)>2.5,8,0) }
EBDEC BDEC VSS VALUE = { IF(V(B0,VSS)>2.5,1,0) + IF(V(B1,VSS)>2.5,2,0) + IF(V(B2,VSS)>2.5,4,0) + IF(V(B3,VSS)>2.5,8,0) }

ELOCGT LOCGT VSS VALUE = { IF(V(ADEC,VSS)-V(BDEC,VSS)>0.5, V(VDD,VSS), 0) }
ELOCLT LOCLT VSS VALUE = { IF(V(BDEC,VSS)-V(ADEC,VSS)>0.5, V(VDD,VSS), 0) }
ELOCEQ LOCEQ VSS VALUE = { IF(ABS(V(ADEC,VSS)-V(BDEC,VSS))<0.5, V(VDD,VSS), 0) }

EOGT OAGTB VSS VALUE = { IF(V(LOCGT,VSS)>2.5, V(VDD,VSS), IF(V(LOCEQ,VSS)>2.5, V(IAGTB,VSS), 0)) }
EOEQ OAEQB VSS VALUE = { IF(V(LOCEQ,VSS)>2.5, V(IAEQB,VSS), 0) }
EOLT OALTB VSS VALUE = { IF(V(LOCLT,VSS)>2.5, V(VDD,VSS), IF(V(LOCEQ,VSS)>2.5, V(IALTB,VSS), 0)) }
.ENDS
```

## Example Usage

```cir
.include "7485.cir"

VDD vdd 0 DC 5
VA3 a3 0 DC 0
VA2 a2 0 DC 1
VA1 a1 0 DC 1
VA0 a0 0 DC 0
VB3 b3 0 DC 0
VB2 b2 0 DC 1
VB1 b1 0 DC 0
VB0 b0 0 DC 1
VGT igt 0 DC 0
VEQ ieq 0 DC 5
VLT ilt 0 DC 0

XU1
+ a3 a2 a1 a0 b3 b2 b1 b0
+ igt ieq ilt
+ ogt oeq olt
+ vdd 0
+ CMOS_7485

.op
.probe V(ogt) V(oeq) V(olt)
.end
```
