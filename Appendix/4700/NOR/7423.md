The `74x23` is the dual 4-input NOR-with-strobe device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7423`
- reusable CMOS helper blocks for inversion, 2-input logic, and 4-input composition
- one packaged device containing 2 strobe-controlled NOR sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the standalone `74x23` function, not an electrical model of a specific TTL or CMOS vendor family.
- The original `7423` provides dedicated expander inputs on gate 1. This simplified CMOS model omits that expansion interface and matches the published TI Boolean function with the expander inputs left open.
- The port order is logical and grouped by gate; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.

## Port Order

```cir
.SUBCKT CMOS_7423
+ A1 B1 C1 D1 G1 Y1
+ A2 B2 C2 D2 G2 Y2
+ VDD VSS
```

## Function

The simplified standalone model implements the following logic function for each section:

$$Y = \neg(G \land (A \lor B \lor C \lor D))$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7423.cir**
```cir
* Functional CMOS 74x23 model.
* The original expander inputs on gate 1 are omitted. This model matches
* the standalone function with the expander pins left open.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NAND2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NOR2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR4 A B C D Y VDD VSS
XO1 A B N1 VDD VSS CMOS_OR2
XO2 C D N2 VDD VSS CMOS_OR2
XO3 N1 N2 Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_STROBED_NOR4 A B C D G Y VDD VSS
XO1 A B C D N1 VDD VSS CMOS_OR4
XA1 G N1 N2 VDD VSS CMOS_AND2
XI1 N2 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_7423
+ A1 B1 C1 D1 G1 Y1
+ A2 B2 C2 D2 G2 Y2
+ VDD VSS

XG1 A1 B1 C1 D1 G1 N1 VDD VSS CMOS_STROBED_NOR4
EYY1 Y1 VSS VALUE = { V(N1,VSS) }
XG2 A2 B2 C2 D2 G2 N2 VDD VSS CMOS_STROBED_NOR4
EYY2 Y2 VSS VALUE = { V(N2,VSS) }
.ENDS
```

## Example Usage

```cir
.include "7423.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1
VB1 b1 0 DC 0
VC1 c1 0 DC 0
VD1 d1 0 DC 0
VG1 g1 0 DC 1

XU1
+ a1 b1 c1 d1 g1 y1
+ 0 0 0 0 0 y2
+ vdd 0
+ CMOS_7423

.op
.probe V(y1)
.end
```
