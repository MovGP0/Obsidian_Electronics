The `74x136` is the quad exclusive-OR device with open-collector outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74136`
- reusable CMOS helper blocks for inversion, buffering, multi-input logic, and open-drain outputs where needed
- one packaged device containing 4 independent logic sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x136` function, not an electrical model of a specific TTL or CMOS vendor family.
- The port order is logical and grouped by gate; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Open-collector outputs are modeled as open-drain pull-down stages. Add an external pull-up resistor in simulations.

## Port Order

```cir
.SUBCKT CMOS_74136
+ A1 B1 Y1 A2 B2 Y2
+ A3 B3 Y3 A4 B4 Y4
+ VDD VSS
```

## Function

The device implements the following logic function:

$$Y = A \oplus B$$

The HIGH state is provided by an external pull-up because the output stage is open-collector.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74136.cir**
```cir
* Functional CMOS 74x136 model.
* Open-collector behavior is represented as an open-drain pull-down stage.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_BUF A Y VDD VSS
XU1 A N1 VDD VSS CMOS_INV
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NAND2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_AND3 A B C Y VDD VSS
XA1 A B N1 VDD VSS CMOS_AND2
XA2 N1 C Y VDD VSS CMOS_AND2
.ENDS

.SUBCKT CMOS_AND4 A B C D Y VDD VSS
XA1 A B N1 VDD VSS CMOS_AND2
XA2 C D N2 VDD VSS CMOS_AND2
XA3 N1 N2 Y VDD VSS CMOS_AND2
.ENDS

.SUBCKT CMOS_NAND3 A B C Y VDD VSS
XA1 A B C N1 VDD VSS CMOS_AND3
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_NAND4 A B C D Y VDD VSS
XA1 A B C D N1 VDD VSS CMOS_AND4
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NOR2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR3 A B C Y VDD VSS
XO1 A B N1 VDD VSS CMOS_OR2
XO2 N1 C Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_OR4 A B C D Y VDD VSS
XO1 A B N1 VDD VSS CMOS_OR2
XO2 C D N2 VDD VSS CMOS_OR2
XO3 N1 N2 Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_NOR3 A B C Y VDD VSS
XO1 A B C N1 VDD VSS CMOS_OR3
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_NOR4 A B C D Y VDD VSS
XO1 A B C D N1 VDD VSS CMOS_OR4
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_XOR2 A B Y VDD VSS
XIA A NA VDD VSS CMOS_INV
XIB B NB VDD VSS CMOS_INV
XA1 NA B T1 VDD VSS CMOS_AND2
XA2 A NB T2 VDD VSS CMOS_AND2
XO1 T1 T2 Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_XNOR2 A B Y VDD VSS
XX1 A B N1 VDD VSS CMOS_XOR2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OD_OUT CTRL Y VSS
MN1 Y CTRL VSS VSS CMOSN L=1U W=2U
.ENDS

.SUBCKT CMOS_74136
+ A1 B1 Y1 A2 B2 Y2
+ A3 B3 Y3 A4 B4 Y4
+ VDD VSS

XG1 A1 B1 CTRL1 VDD VSS CMOS_XNOR2
XOD1 CTRL1 Y1 VSS CMOS_OD_OUT
XG2 A2 B2 CTRL2 VDD VSS CMOS_XNOR2
XOD2 CTRL2 Y2 VSS CMOS_OD_OUT
XG3 A3 B3 CTRL3 VDD VSS CMOS_XNOR2
XOD3 CTRL3 Y3 VSS CMOS_OD_OUT
XG4 A4 B4 CTRL4 VDD VSS CMOS_XNOR2
XOD4 CTRL4 Y4 VSS CMOS_OD_OUT
.ENDS
```

## Example Usage

```cir
.include "74136.cir"

VDD vdd 0 DC 5
RPULL1 y1 vdd 4.7K
VA1 a1 0 DC 1
VB1 b1 0 DC 0

XU1
+ a1 b1 y1 0 0 y2 0 0 y3 0 0 y4 vdd 0
+ CMOS_74136

.op
.probe V(y1)
.end
```
