The `74x802` is the triple 4-input OR/NOR driver with complementary outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74802`
- reusable CMOS helper blocks for inversion, 2-input logic, and 4-input composition
- one packaged device containing 3 independent 4-input OR/NOR sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x802` function, not an electrical model of a specific TTL or CMOS vendor family.
- Each section provides complementary outputs: an OR output and a NOR output.
- The port order is logical and grouped by section; it does not try to reproduce the physical multi-ground package pin numbering used by historical parts.
- Push-pull outputs are modeled as ordinary CMOS outputs.
- Driver-strength details and exact TTL line-drive behavior are intentionally not modeled.

## Port Order

```cir
.SUBCKT CMOS_74802
+ A1 B1 C1 D1 Y1 Z1
+ A2 B2 C2 D2 Y2 Z2
+ A3 B3 C3 D3 Y3 Z3
+ VDD VSS
```

## Function

For each section, the device provides complementary logic outputs:

$$Y = A \lor B \lor C \lor D$$

$$Z = \neg(A \lor B \lor C \lor D)$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74802.cir**
```cir
* Functional CMOS 74x802 model.
* Each section provides complementary OR and NOR outputs.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NOR2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR4 A B C D Y VDD VSS
XO1 A B N1 VDD VSS CMOS_OR2
XO2 C D N2 VDD VSS CMOS_OR2
XO3 N1 N2 Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_74802
+ A1 B1 C1 D1 Y1 Z1
+ A2 B2 C2 D2 Y2 Z2
+ A3 B3 C3 D3 Y3 Z3
+ VDD VSS

XG1 A1 B1 C1 D1 N1 VDD VSS CMOS_OR4
EYY1 Y1 VSS VALUE = { V(N1,VSS) }
XI1 N1 N2 VDD VSS CMOS_INV
EZZ1 Z1 VSS VALUE = { V(N2,VSS) }

XG2 A2 B2 C2 D2 N3 VDD VSS CMOS_OR4
EYY2 Y2 VSS VALUE = { V(N3,VSS) }
XI2 N3 N4 VDD VSS CMOS_INV
EZZ2 Z2 VSS VALUE = { V(N4,VSS) }

XG3 A3 B3 C3 D3 N5 VDD VSS CMOS_OR4
EYY3 Y3 VSS VALUE = { V(N5,VSS) }
XI3 N5 N6 VDD VSS CMOS_INV
EZZ3 Z3 VSS VALUE = { V(N6,VSS) }
.ENDS
```

## Example Usage

```cir
.include "74802.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1
VB1 b1 0 DC 0
VC1 c1 0 DC 0
VD1 d1 0 DC 0

XU1
+ a1 b1 c1 d1 y1 z1
+ 0 0 0 0 y2 z2
+ 0 0 0 0 y3 z3
+ vdd 0
+ CMOS_74802

.op
.probe V(y1) V(z1)
.end
```
