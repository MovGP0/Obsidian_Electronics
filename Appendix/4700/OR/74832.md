The `74x832` is the hex 2-input OR driver in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74832`
- reusable CMOS helper blocks for inversion and 2-input OR logic
- one packaged device containing 6 independent OR sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x832` function, not an electrical model of a specific TTL or CMOS vendor family.
- The port order is logical and grouped by gate; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.
- Driver-strength details and capacitive-load behavior are intentionally not modeled.

## Port Order

```cir
.SUBCKT CMOS_74832
+ A1 B1 Y1 A2 B2 Y2
+ A3 B3 Y3 A4 B4 Y4
+ A5 B5 Y5 A6 B6 Y6
+ VDD VSS
```

## Function

For each section, the device implements the following logic function:

$$Y = A \lor B$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74832.cir**
```cir
* Functional CMOS 74x832 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NOR2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_74832
+ A1 B1 Y1 A2 B2 Y2
+ A3 B3 Y3 A4 B4 Y4
+ A5 B5 Y5 A6 B6 Y6
+ VDD VSS

XG1 A1 B1 N1 VDD VSS CMOS_OR2
EYY1 Y1 VSS VALUE = { V(N1,VSS) }
XG2 A2 B2 N2 VDD VSS CMOS_OR2
EYY2 Y2 VSS VALUE = { V(N2,VSS) }
XG3 A3 B3 N3 VDD VSS CMOS_OR2
EYY3 Y3 VSS VALUE = { V(N3,VSS) }
XG4 A4 B4 N4 VDD VSS CMOS_OR2
EYY4 Y4 VSS VALUE = { V(N4,VSS) }
XG5 A5 B5 N5 VDD VSS CMOS_OR2
EYY5 Y5 VSS VALUE = { V(N5,VSS) }
XG6 A6 B6 N6 VDD VSS CMOS_OR2
EYY6 Y6 VSS VALUE = { V(N6,VSS) }
.ENDS
```

## Example Usage

```cir
.include "74832.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1
VB1 b1 0 DC 0

XU1
+ a1 b1 y1 0 0 y2 0 0 y3
+ 0 0 y4 0 0 y5 0 0 y6
+ vdd 0
+ CMOS_74832

.op
.probe V(y1)
.end
```
