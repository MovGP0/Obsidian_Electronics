The `74x77` is a quad D latch variant in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7477`
- reusable CMOS helper blocks for inversion and level-sensitive D latching
- one packaged device containing 4 transparent D latches arranged as 2 enable pairs

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x77` latch function, not an electrical model of a specific TTL or CMOS vendor family.
- This simplified CMOS note follows the common `74x77` family distinction of exposing only the true outputs while keeping the `74x75`-style latch behavior.
- Latches 1 and 2 share one active-high enable input, and latches 3 and 4 share a second active-high enable input.
- The port order is logical and grouped by latch pair; it does not try to reproduce the physical package arrangement.

## Port Order

```cir
.SUBCKT CMOS_7477
+ C12 D1 Q1 D2 Q2
+ C34 D3 Q3 D4 Q4
+ VDD VSS
```

## Function

For each latch:

- if `C = 1`, then `Q = D`
- if `C = 0`, then `Q` holds its previous state

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7477.cir**
```cir
* Functional CMOS 74x77 model.
* The simplified model exposes only the true outputs.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_DLAT_H_Q D EN Q VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QB VDD VSS CMOS_INV
XI2 QB Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_7477
+ C12 D1 Q1 D2 Q2
+ C34 D3 Q3 D4 Q4
+ VDD VSS

XL1 D1 C12 Q1 VDD VSS CMOS_DLAT_H_Q
XL2 D2 C12 Q2 VDD VSS CMOS_DLAT_H_Q
XL3 D3 C34 Q3 VDD VSS CMOS_DLAT_H_Q
XL4 D4 C34 Q4 VDD VSS CMOS_DLAT_H_Q
.ENDS
```

## Example Usage

```cir
.include "7477.cir"

VDD vdd 0 DC 5
VC12 c12 0 PULSE(0 5 0 1N 1N 50N 100N)
VC34 c34 0 DC 0
VD1 d1 0 PULSE(0 5 0 1N 1N 20N 40N)

XU1
+ c12 d1 q1 0 q2
+ c34 0 q3 0 q4
+ vdd 0
+ CMOS_7477

.tran 1N 200N
.probe V(q1)
.end
```
