The `74x279` is the quadruple active-low S-R latch device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74279`
- reusable CMOS helper blocks for inversion, NAND logic, and cross-coupled S-R storage
- one packaged device containing 4 active-low S-R latches

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x279` latch function, not an electrical model of a specific TTL or CMOS vendor family.
- Latches 1 and 3 expose the dual active-low set inputs found in the TTL package. Latches 2 and 4 use one active-low set input each.
- Inputs are active low: pulsing a set input LOW drives `Q` HIGH; pulsing the reset input LOW drives `Q` LOW.
- Simultaneously asserting set and reset for the same latch is treated as undefined, matching the usual device-level guidance.
- The port order is logical and grouped by latch.

## Port Order

```cir
.SUBCKT CMOS_74279
+ R1N S1A_N S1B_N Q1
+ R2N S2N Q2
+ R3N S3A_N S3B_N Q3
+ R4N S4N Q4
+ VDD VSS
```

## Function

For each latch:

- if any `S_N = 0`, then `Q = 1`
- else if `R_N = 0`, then `Q = 0`
- else `Q` holds its previous state

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74279.cir**
```cir
* Functional CMOS 74x279 model.
* Latches 1 and 3 include the dual set-input structure used by the TTL device.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XN1 A B N1 VDD VSS CMOS_NAND2
XI1 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_SR_LATCH_N RN SN Q VDD VSS
XN1 SN QB Q VDD VSS CMOS_NAND2
XN2 RN Q QB VDD VSS CMOS_NAND2
.ENDS

.SUBCKT CMOS_SR_LATCH_N2 RN S1N S2N Q VDD VSS
XA1 S1N S2N SN VDD VSS CMOS_AND2
XL1 RN SN Q VDD VSS CMOS_SR_LATCH_N
.ENDS

.SUBCKT CMOS_74279
+ R1N S1A_N S1B_N Q1
+ R2N S2N Q2
+ R3N S3A_N S3B_N Q3
+ R4N S4N Q4
+ VDD VSS

XL1 R1N S1A_N S1B_N Q1 VDD VSS CMOS_SR_LATCH_N2
XL2 R2N S2N Q2 VDD VSS CMOS_SR_LATCH_N
XL3 R3N S3A_N S3B_N Q3 VDD VSS CMOS_SR_LATCH_N2
XL4 R4N S4N Q4 VDD VSS CMOS_SR_LATCH_N
.ENDS
```

## Example Usage

```cir
.include "74279.cir"

VDD vdd 0 DC 5
VR1 r1n 0 DC 1
VS1A s1an 0 DC 1
VS1B s1bn 0 PULSE(5 0 0 1N 1N 10N 80N)

XU1
+ r1n s1an s1bn q1
+ 1 1 q2
+ 1 1 1 q3
+ 1 1 q4
+ vdd 0
+ CMOS_74279

.tran 1N 120N
.probe V(q1)
.end
```
