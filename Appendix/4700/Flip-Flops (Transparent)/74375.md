The `74x375` is the quad D latch with complementary outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74375`
- reusable CMOS helper blocks for inversion and level-sensitive D latching
- one packaged device containing 4 transparent D latches arranged as 2 enable pairs

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x375` latch function, not an electrical model of a specific TTL or CMOS vendor family.
- The `74x375` is treated as functionally identical to the `74x75`; only terminal arrangement differs in the classic TTL family.
- Latches 1 and 2 share one active-high enable input, and latches 3 and 4 share a second active-high enable input.
- Both true and complement outputs are provided for each latch.
- The port order is logical and grouped by latch pair; it does not reproduce the alternate package pinout.

## Port Order

```cir
.SUBCKT CMOS_74375
+ C12 D1 Q1 Q1N D2 Q2 Q2N
+ C34 D3 Q3 Q3N D4 Q4 Q4N
+ VDD VSS
```

## Function

For each latch:

- if `C = 1`, then `Q = D`
- if `C = 0`, then `Q` holds its previous state
- `Q_N = not(Q)`

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74375.cir**
```cir
* Functional CMOS 74x375 model.
* This simplified model uses the same latch behavior as the 74x75 family.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_DLAT_H D EN Q QN VDD VSS
S1 NSTORE D EN VSS SWEN
C1 NSTORE VSS 1P
R1 NSTORE VSS 1T
XI1 NSTORE QN VDD VSS CMOS_INV
XI2 QN Q VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_74375
+ C12 D1 Q1 Q1N D2 Q2 Q2N
+ C34 D3 Q3 Q3N D4 Q4 Q4N
+ VDD VSS

XL1 D1 C12 Q1 Q1N VDD VSS CMOS_DLAT_H
XL2 D2 C12 Q2 Q2N VDD VSS CMOS_DLAT_H
XL3 D3 C34 Q3 Q3N VDD VSS CMOS_DLAT_H
XL4 D4 C34 Q4 Q4N VDD VSS CMOS_DLAT_H
.ENDS
```

## Example Usage

```cir
.include "74375.cir"

VDD vdd 0 DC 5
VC12 c12 0 PULSE(0 5 0 1N 1N 50N 100N)
VD1 d1 0 PULSE(0 5 0 1N 1N 20N 40N)

XU1
+ c12 d1 q1 q1n 0 q2 q2n
+ 0 0 q3 q3n 0 q4 q4n
+ vdd 0
+ CMOS_74375

.tran 1N 200N
.probe V(q1) V(q1n)
.end
```
