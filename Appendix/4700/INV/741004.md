The `74x1004` is the buffered version of the 74x04 inverter function in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_741004`
- reusable CMOS helper blocks for inversion, buffering, multi-input logic, and open-drain outputs where needed
- one packaged device containing 6 independent logic sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x1004` function, not an electrical model of a specific TTL or CMOS vendor family.
- The port order is logical and grouped by gate; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.
- Buffer or driver variants keep the same Boolean function; the model does not attempt to reproduce stronger electrical drive or fan-out.

## Port Order

```cir
.SUBCKT CMOS_741004
+ A1 Y1 A2 Y2 A3 Y3
+ A4 Y4 A5 Y5 A6 Y6
+ VDD VSS
```

## Function

$$Y = \neg(A)$$

## Example Usage

```cir
.include "741004.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1

XU1
+ a1 y1 0 y2 0 y3 0 y4 0 y5 0 y6 vdd 0
+ CMOS_741004

.op
.probe V(y1)
.end
```
