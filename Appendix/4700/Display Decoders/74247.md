The `74x247` is the BCD-to-seven-segment decoder/driver for LED displays in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74247`
- a behavioral BCD-to-seven-segment decoder using open-collector outputs
- simplified ripple-blanking and lamp-test behavior for one packaged device

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x247` function, not an electrical model of a specific TTL or CMOS vendor family.
- The original TTL `BI/RBO` multifunction node is split here into separate logical `BI_N` and `RBO_N` pins so the blanking chain is easier to simulate.
- Outputs are active-low open-collector segment drivers intended for common-anode LED displays and therefore require external pull-up or LED load paths.
- `LT_N` forces all segments on, `BI_N` blanks the display, and `RBI_N` suppresses a decoded zero and drives `RBO_N` low for ripple blanking.
- This simplified model decodes valid BCD digits `0` through `9`. Inputs `10` through `15` are blanked instead of reproducing the legacy non-decimal glyphs.
- The historical `74x247` font variant with tailed `6` and `9` is preserved in the segment decode.

## Port Order

```cir
.SUBCKT CMOS_74247
+ D C B A LTN RBIN BIN RBO
+ AN BN CN DN EN FN GN
+ VDD VSS
```

## Function

- if `BI_N = 0`, all segment outputs are off
- else if `LT_N = 0`, all segment outputs are on
- else if `RBI_N = 0` and the BCD input is `0000`, the digit is blanked and `RBO_N = 0`
- else the BCD input is decoded to active-low segment outputs for digits `0` through `9`

## Netlist

> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

## Example Usage

```cir
.include "74247.cir"

VDD vdd 0 DC 5
RSEG_A an vdd 1K
RSEG_B bn vdd 1K
RSEG_C cn vdd 1K
RSEG_D dn vdd 1K
RSEG_E en vdd 1K
RSEG_F fn vdd 1K
RSEG_G gn vdd 1K

VD d 0 DC 1
VC c 0 DC 0
VB b 0 DC 0
VA a 0 DC 1
VLT ltn 0 DC 5
VRBI rbin 0 DC 5
VBI bin 0 DC 5

XU1
+ d c b a ltn rbin bin rbo
+ an bn cn dn en fn gn
+ vdd 0
+ CMOS_74247

.op
.probe V(an) V(bn) V(cn) V(dn) V(en) V(fn) V(gn)
.end
```
