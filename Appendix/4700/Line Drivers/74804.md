The `74x804` is the hex 2-input NAND line-driver device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74804`
- reusable CMOS helper blocks for inversion and 2-input NAND logic
- one packaged device containing 6 independent line-driver sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x804` function, not an electrical model of a specific TTL or CMOS vendor family.
- Line-driver variants keep the same Boolean function; the model does not attempt to reproduce higher current drive or transmission-line behavior.
- The port order is logical and grouped by gate; it does not try to reproduce the physical package pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.

## Port Order

```cir
.SUBCKT CMOS_74804
+ A1 B1 Y1 A2 B2 Y2
+ A3 B3 Y3 A4 B4 Y4
+ A5 B5 Y5 A6 B6 Y6
+ VDD VSS
```

## Function

$$Y = \neg(A \land B)$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74804.cir**
```cir
* Functional CMOS 74x804 model.
* Line-driver variants keep the same logic function but do not model stronger drive.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_74804
+ A1 B1 Y1 A2 B2 Y2
+ A3 B3 Y3 A4 B4 Y4
+ A5 B5 Y5 A6 B6 Y6
+ VDD VSS

XG1 A1 B1 N1 VDD VSS CMOS_NAND2
EYY1 Y1 VSS VALUE = { V(N1,VSS) }
XG2 A2 B2 N2 VDD VSS CMOS_NAND2
EYY2 Y2 VSS VALUE = { V(N2,VSS) }
XG3 A3 B3 N3 VDD VSS CMOS_NAND2
EYY3 Y3 VSS VALUE = { V(N3,VSS) }
XG4 A4 B4 N4 VDD VSS CMOS_NAND2
EYY4 Y4 VSS VALUE = { V(N4,VSS) }
XG5 A5 B5 N5 VDD VSS CMOS_NAND2
EYY5 Y5 VSS VALUE = { V(N5,VSS) }
XG6 A6 B6 N6 VDD VSS CMOS_NAND2
EYY6 Y6 VSS VALUE = { V(N6,VSS) }
.ENDS
```

## Example Usage

```cir
.include "74804.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1
VB1 b1 0 DC 0

XU1
+ a1 b1 y1 a2 b2 y2
+ a3 b3 y3 a4 b4 y4
+ a5 b5 y5 a6 b6 y6
+ vdd 0
+ CMOS_74804

.op
.probe V(y1)
.end
```
