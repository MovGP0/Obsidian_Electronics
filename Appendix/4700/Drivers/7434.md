The `74x34` is the hex noninverting buffer device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7434`
- reusable CMOS helper blocks for inversion and noninverting buffering
- one packaged device containing 6 independent buffer sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x34` function, not an electrical model of a specific TTL or CMOS vendor family.
- The port order is logical and grouped by buffer; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.

## Port Order

```cir
.SUBCKT CMOS_7434
+ A1 Y1 A2 Y2 A3 Y3
+ A4 Y4 A5 Y5 A6 Y6
+ VDD VSS
```

## Function

$$Y = A$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7434.cir**
```cir
* Functional CMOS 74x34 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_BUF A Y VDD VSS
XU1 A N1 VDD VSS CMOS_INV
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_7434
+ A1 Y1 A2 Y2 A3 Y3
+ A4 Y4 A5 Y5 A6 Y6
+ VDD VSS

XG1 A1 N1 VDD VSS CMOS_BUF
EYY1 Y1 VSS VALUE = { V(N1,VSS) }
XG2 A2 N2 VDD VSS CMOS_BUF
EYY2 Y2 VSS VALUE = { V(N2,VSS) }
XG3 A3 N3 VDD VSS CMOS_BUF
EYY3 Y3 VSS VALUE = { V(N3,VSS) }
XG4 A4 N4 VDD VSS CMOS_BUF
EYY4 Y4 VSS VALUE = { V(N4,VSS) }
XG5 A5 N5 VDD VSS CMOS_BUF
EYY5 Y5 VSS VALUE = { V(N5,VSS) }
XG6 A6 N6 VDD VSS CMOS_BUF
EYY6 Y6 VSS VALUE = { V(N6,VSS) }
.ENDS
```

## Example Usage

```cir
.include "7434.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1

XU1
+ a1 y1 0 y2 0 y3
+ 0 y4 0 y5 0 y6
+ vdd 0
+ CMOS_7434

.op
.probe V(y1)
.end
```
