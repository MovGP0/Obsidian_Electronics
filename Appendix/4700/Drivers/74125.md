The `74x125` is the quad noninverting buffer device with 3-state outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74125`
- reusable CMOS helper blocks for inversion, noninverting buffering, and tri-state output control
- one packaged device containing 4 independent buffer sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x125` function, not an electrical model of a specific TTL or CMOS vendor family.
- Each section has an active-low enable input. When disabled, the output is high-impedance.
- The port order is logical and grouped by section; it does not try to reproduce the physical DIP pin numbering.
- Push-pull enabled outputs are modeled as ordinary CMOS outputs. The disabled state is modeled as high-impedance.

## Port Order

```cir
.SUBCKT CMOS_74125
+ G1N A1 Y1 G2N A2 Y2
+ G3N A3 Y3 G4N A4 Y4
+ VDD VSS
```

## Function

For each section:

- if `G_N = 0`, then `Y = A`
- if `G_N = 1`, then `Y = Z`

where `Z` is the high-impedance state.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74125.cir**
```cir
* Functional CMOS 74x125 model.
* Outputs enter a high-impedance state when the active-low enable input is HIGH.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_TS_BUF_H A EN Y VDD VSS
EIN DIN VSS VALUE = { V(A,VSS) }
S1 Y DIN EN VSS SWEN
.ENDS

.SUBCKT CMOS_TS_BUF_L A ENN Y VDD VSS
XI1 ENN EN VDD VSS CMOS_INV
XT1 A EN Y VDD VSS CMOS_TS_BUF_H
.ENDS

.SUBCKT CMOS_74125
+ G1N A1 Y1 G2N A2 Y2
+ G3N A3 Y3 G4N A4 Y4
+ VDD VSS

XG1 A1 G1N Y1 VDD VSS CMOS_TS_BUF_L
XG2 A2 G2N Y2 VDD VSS CMOS_TS_BUF_L
XG3 A3 G3N Y3 VDD VSS CMOS_TS_BUF_L
XG4 A4 G4N Y4 VDD VSS CMOS_TS_BUF_L
.ENDS
```

## Example Usage

```cir
.include "74125.cir"

VDD vdd 0 DC 5
RLOAD y1 0 100K
VG1 g1n 0 DC 1
VA1 a1 0 DC 1

XU1
+ g1n a1 y1 vdd 0 y2
+ vdd 0 y3 vdd 0 y4
+ vdd 0
+ CMOS_74125

.op
.probe V(y1)
.end
```
