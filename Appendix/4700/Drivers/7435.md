The `74x35` is the hex noninverting buffer device with open-collector outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7435`
- reusable CMOS helper blocks for inversion and open-drain output modeling
- one packaged device containing 6 independent buffer sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x35` function, not an electrical model of a specific TTL or CMOS vendor family.
- The port order is logical and grouped by buffer; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Open-collector outputs are modeled as open-drain pull-down stages. Add an external pull-up resistor in simulations.

## Port Order

```cir
.SUBCKT CMOS_7435
+ A1 Y1 A2 Y2 A3 Y3
+ A4 Y4 A5 Y5 A6 Y6
+ VDD VSS
```

## Function

The device implements the following logic function:

$$Y = A$$

The HIGH state is provided by an external pull-up because the output stage is open-collector.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7435.cir**
```cir
* Functional CMOS 74x35 model.
* Open-collector behavior is represented as an open-drain pull-down stage.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_OD_OUT CTRL Y VSS
MN1 Y CTRL VSS VSS CMOSN L=1U W=2U
.ENDS

.SUBCKT CMOS_7435
+ A1 Y1 A2 Y2 A3 Y3
+ A4 Y4 A5 Y5 A6 Y6
+ VDD VSS

XI1 A1 C1 VDD VSS CMOS_INV
XOD1 C1 Y1 VSS CMOS_OD_OUT
XI2 A2 C2 VDD VSS CMOS_INV
XOD2 C2 Y2 VSS CMOS_OD_OUT
XI3 A3 C3 VDD VSS CMOS_INV
XOD3 C3 Y3 VSS CMOS_OD_OUT
XI4 A4 C4 VDD VSS CMOS_INV
XOD4 C4 Y4 VSS CMOS_OD_OUT
XI5 A5 C5 VDD VSS CMOS_INV
XOD5 C5 Y5 VSS CMOS_OD_OUT
XI6 A6 C6 VDD VSS CMOS_INV
XOD6 C6 Y6 VSS CMOS_OD_OUT
.ENDS
```

## Example Usage

```cir
.include "7435.cir"

VDD vdd 0 DC 5
RPULL1 y1 vdd 4.7K
VA1 a1 0 DC 1

XU1
+ a1 y1 0 y2 0 y3
+ 0 y4 0 y5 0 y6
+ vdd 0
+ CMOS_7435

.op
.probe V(y1)
.end
```
