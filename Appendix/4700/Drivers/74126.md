The `74x126` is the quad noninverting buffer device with 3-state outputs in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74126`
- reusable CMOS helper blocks for inversion, noninverting buffering, and tri-state output control
- one packaged device containing 4 independent buffer sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x126` function, not an electrical model of a specific TTL or CMOS vendor family.
- Each section has an active-high enable input. When disabled, the output is high-impedance.
- The port order is logical and grouped by section; it does not try to reproduce the physical DIP pin numbering.
- Push-pull enabled outputs are modeled as ordinary CMOS outputs. The disabled state is modeled as high-impedance.

## Port Order

```cir
.SUBCKT CMOS_74126
+ G1 A1 Y1 G2 A2 Y2
+ G3 A3 Y3 G4 A4 Y4
+ VDD VSS
```

## Function

For each section:

- if `G = 1`, then `Y = A`
- if `G = 0`, then `Y = Z`

where `Z` is the high-impedance state.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74126.cir**
```cir
* Functional CMOS 74x126 model.
* Outputs enter a high-impedance state when the enable input is LOW.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL SWEN SW VT=2.5 VH=0.1 RON=10 ROFF=1E12

.SUBCKT CMOS_TS_BUF_H A EN Y VDD VSS
EIN DIN VSS VALUE = { V(A,VSS) }
S1 Y DIN EN VSS SWEN
.ENDS

.SUBCKT CMOS_74126
+ G1 A1 Y1 G2 A2 Y2
+ G3 A3 Y3 G4 A4 Y4
+ VDD VSS

XG1 A1 G1 Y1 VDD VSS CMOS_TS_BUF_H
XG2 A2 G2 Y2 VDD VSS CMOS_TS_BUF_H
XG3 A3 G3 Y3 VDD VSS CMOS_TS_BUF_H
XG4 A4 G4 Y4 VDD VSS CMOS_TS_BUF_H
.ENDS
```

## Example Usage

```cir
.include "74126.cir"

VDD vdd 0 DC 5
RLOAD y1 0 100K
VG1 g1 0 DC 0
VA1 a1 0 DC 1

XU1
+ g1 a1 y1 0 0 y2
+ 0 0 y3 0 0 y4
+ vdd 0
+ CMOS_74126

.op
.probe V(y1)
.end
```
