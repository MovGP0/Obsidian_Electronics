The `74x280` is the 9-bit odd/even parity generator/checker in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_74280`
- behavioral parity-reduction logic for 9 data inputs
- one packaged device with direct odd/even parity outputs

## Model Scope

- The model is a functionally equivalent CMOS implementation of the `74x280` function, not an electrical model of a specific TTL or CMOS vendor family.
- Unlike the older `74x180`, this simplified note treats the device as a direct 9-input parity generator/checker with odd and even outputs only.
- The legacy TI documentation notes that the `74x280` can replace many `74x180` uses because of its compatible parity function, despite different internal pin usage.
- The outputs are complementary parity indicators:
  - `EOUT = 1` when the 9 input bits contain an even number of ones
  - `OOUT = 1` when the 9 input bits contain an odd number of ones
- The port order is logical and grouped by data and result signals.

## Port Order

```cir
.SUBCKT CMOS_74280
+ A B C D E F G H I
+ EOUT OOUT
+ VDD VSS
```

## Function

Let

`P = A xor B xor C xor D xor E xor F xor G xor H xor I`

Then:

- `EOUT = not(P)`
- `OOUT = P`

These outputs can be used either to generate a parity bit or to check an incoming 9-bit word.

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**74280.cir**
```cir
* Functional CMOS 74x280 model.

.SUBCKT CMOS_74280
+ A B C D E F G H I
+ EOUT OOUT
+ VDD VSS

EPAR PAR VSS VALUE = {
+ MOD(
+   IF(V(A,VSS)>2.5,1,0) + IF(V(B,VSS)>2.5,1,0) +
+   IF(V(C,VSS)>2.5,1,0) + IF(V(D,VSS)>2.5,1,0) +
+   IF(V(E,VSS)>2.5,1,0) + IF(V(F,VSS)>2.5,1,0) +
+   IF(V(G,VSS)>2.5,1,0) + IF(V(H,VSS)>2.5,1,0) +
+   IF(V(I,VSS)>2.5,1,0),
+   2)
+ }

EEOUT EOUT VSS VALUE = { IF(V(PAR,VSS)<0.5, V(VDD,VSS), 0) }
EOOUT OOUT VSS VALUE = { IF(V(PAR,VSS)>0.5, V(VDD,VSS), 0) }
.ENDS
```

## Example Usage

```cir
.include "74280.cir"

VDD vdd 0 DC 5
VA a 0 DC 1
VB b 0 DC 0
VC c 0 DC 1
VD d 0 DC 1
VE e 0 DC 0
VF f 0 DC 0
VG g 0 DC 1
VH h 0 DC 0
VI i 0 DC 1

XU1
+ a b c d e f g h i
+ eout oout
+ vdd 0
+ CMOS_74280

.op
.probe V(eout) V(oout)
.end
```
