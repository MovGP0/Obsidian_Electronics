The `74x54` is the single 4-wide 2-input AND-OR-INVERT device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7454`
- reusable CMOS helper blocks for inversion, 2-input logic, and AOI composition
- one packaged device containing 1 four-wide AND-OR-INVERT function

## Model Scope

- The model is a functionally equivalent CMOS implementation of the original `7454` function, not an electrical model of a specific TTL or CMOS vendor family.
- Later variants such as `74LS54` regroup the inputs as `2-3-3-2`. This generic `74x54` note models the classic `4 x 2-input` form listed for function `54`.
- The port order is logical and grouped by function; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.

## Port Order

```cir
.SUBCKT CMOS_7454
+ A B C D E F G H
+ Y VDD VSS
```

## Function

$$Y = \neg((A \land B) \lor (C \land D) \lor (E \land F) \lor (G \land H))$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7454.cir**
```cir
* Functional CMOS 74x54 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NAND2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NOR2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR4 A B C D Y VDD VSS
XO1 A B N1 VDD VSS CMOS_OR2
XO2 C D N2 VDD VSS CMOS_OR2
XO3 N1 N2 Y VDD VSS CMOS_OR2
.ENDS

.SUBCKT CMOS_AOI42 A B C D E F G H Y VDD VSS
XA1 A B N1 VDD VSS CMOS_AND2
XA2 C D N2 VDD VSS CMOS_AND2
XA3 E F N3 VDD VSS CMOS_AND2
XA4 G H N4 VDD VSS CMOS_AND2
XO1 N1 N2 N3 N4 N5 VDD VSS CMOS_OR4
XI1 N5 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_7454
+ A B C D E F G H
+ Y VDD VSS

XG1 A B C D E F G H N1 VDD VSS CMOS_AOI42
EYY Y VSS VALUE = { V(N1,VSS) }
.ENDS
```

## Example Usage

```cir
.include "7454.cir"

VDD vdd 0 DC 5
VA a 0 DC 1
VB b 0 DC 1
VC c 0 DC 0
VD d 0 DC 0
VE e 0 DC 0
VF f 0 DC 0
VG g 0 DC 0
VH h 0 DC 0

XU1
+ a b c d e f g h y vdd 0
+ CMOS_7454

.op
.probe V(y)
.end
```
