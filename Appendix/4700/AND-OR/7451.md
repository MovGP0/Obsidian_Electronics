The `74x51` is the dual 2-wide 2-input AND-OR-INVERT device in the 74xx logic series.

This note documents a **functionally equivalent CMOS PSpice model** for the package-level device rather than the original TTL transistor network.

## What Is Included

- a top-level `.SUBCKT` named `CMOS_7451`
- reusable CMOS helper blocks for inversion, 2-input logic, and AOI composition
- one packaged device containing 2 independent AND-OR-INVERT sections

## Model Scope

- The model is a functionally equivalent CMOS implementation of the original `7451` / `74S51` function, not an electrical model of a specific TTL or CMOS vendor family.
- Later variants such as `74LS51` regroup the inputs differently. This generic `74x51` note models the classic 2-wide 2-input form listed for the `51` function number.
- The port order is logical and grouped by gate; it does not try to reproduce the physical DIP pin numbering.
- Inputs and outputs are active-high in the logic sense used in the truth tables.
- Push-pull outputs are modeled as ordinary CMOS outputs.

## Port Order

```cir
.SUBCKT CMOS_7451
+ A1 B1 C1 D1 Y1
+ A2 B2 C2 D2 Y2
+ VDD VSS
```

## Function

For each section, the device implements the following logic function:

$$Y = \neg((A \land B) \lor (C \land D))$$

## Netlist

> [!note]
> This model is **functionally equivalent**, not transistor-accurate to the original TTL family member.

**7451.cir**
```cir
* Functional CMOS 74x51 model.

.MODEL CMOSN NMOS LEVEL=1 VTO=0.7 KP=120U GAMMA=0.4 PHI=0.7 LAMBDA=0.02
.MODEL CMOSP PMOS LEVEL=1 VTO=-0.7 KP=60U GAMMA=0.4 PHI=0.7 LAMBDA=0.02

.SUBCKT CMOS_INV A Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NAND2 A B Y VDD VSS
MP1 Y A VDD VDD CMOSP L=1U W=2U
MP2 Y B VDD VDD CMOSP L=1U W=2U
MN1 Y A N1 VSS CMOSN L=1U W=1U
MN2 N1 B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_NOR2 A B Y VDD VSS
MP1 Y A N1 VDD CMOSP L=1U W=2U
MP2 N1 B VDD VDD CMOSP L=1U W=2U
MN1 Y A VSS VSS CMOSN L=1U W=1U
MN2 Y B VSS VSS CMOSN L=1U W=1U
.ENDS

.SUBCKT CMOS_AND2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NAND2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_OR2 A B Y VDD VSS
XU1 A B N1 VDD VSS CMOS_NOR2
XU2 N1 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_AOI22 A B C D Y VDD VSS
XA1 A B N1 VDD VSS CMOS_AND2
XA2 C D N2 VDD VSS CMOS_AND2
XO1 N1 N2 N3 VDD VSS CMOS_OR2
XI1 N3 Y VDD VSS CMOS_INV
.ENDS

.SUBCKT CMOS_7451
+ A1 B1 C1 D1 Y1
+ A2 B2 C2 D2 Y2
+ VDD VSS

XG1 A1 B1 C1 D1 N1 VDD VSS CMOS_AOI22
EYY1 Y1 VSS VALUE = { V(N1,VSS) }
XG2 A2 B2 C2 D2 N2 VDD VSS CMOS_AOI22
EYY2 Y2 VSS VALUE = { V(N2,VSS) }
.ENDS
```

## Example Usage

```cir
.include "7451.cir"

VDD vdd 0 DC 5
VA1 a1 0 DC 1
VB1 b1 0 DC 1
VC1 c1 0 DC 0
VD1 d1 0 DC 0

XU1
+ a1 b1 c1 d1 y1
+ 0 0 0 0 y2
+ vdd 0
+ CMOS_7451

.op
.probe V(y1)
.end
```
