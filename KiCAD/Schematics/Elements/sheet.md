---
title: Sheets
---
**Sheets** place another schematic file as a block in the current schematic. Sheet pins define the electrical interface to the child sheet.
```lisp
(sheet
	(at 50.8 50.8)
	(size 40.64 25.4)
	(stroke
		(width 0.1524)
		(type solid)
	)
	(fill
		(color 0 0 0 0.0000)
	)
	(uuid "11111111-1111-1111-1111-111111111111")
	(property "Sheetname" "CMOS_INV"
		(at 50.8 48.26 0)
	)
	(property "Sheetfile" "cmos_inv.kicad_sch"
		(at 50.8 76.2 0)
	)
	(pin "A" input
		(at 50.8 58.42 180)
		(effects
			(font
				(size 1.27 1.27)
			)
		)
		(uuid "22222222-2222-2222-2222-222222222222")
	)
)
```

## Sheet element

| Field               | Type    | Description                                      |
| ------------------- | ------- | ------------------------------------------------ |
| [[at]]        | point   | Sheet position and optional rotation             |
| `size`              | size    | Sheet rectangle width and height                 |
| `fields_autoplaced` | boolean | Optional KiCad-managed property placement flag   |
| `stroke`            | node    | Sheet border style                               |
| `fill`              | node    | Sheet fill style                                 |
| `uuid`              | uuid    | Unique identifier for this sheet instance        |
| [[property]] | node    | Sheet properties, especially `"Sheetname"` and `"Sheetfile"` |
| `pin`               | node    | Sheet interface pin                              |
| [[sheet_instances]] | node    | Project and path data generated by KiCad         |

## Required properties

| Property      | Description                                        |
| ------------- | -------------------------------------------------- |
| `"Sheetname"` | Visible logical name of the sheet instance         |
| `"Sheetfile"` | Child `.kicad_sch` file referenced by this sheet   |

## Sheet pins

Sheet pins connect parent schematic wires to matching [[label|`hierarchical_label`]] elements inside the child schematic.

| Field     | Type   | Description                                               |
| --------- | ------ | --------------------------------------------------------- |
| name      | string | Sheet pin name; must match a child hierarchical label     |
| type      | enum   | `input`, `output`, `bidirectional`, `tri_state`, or `passive` |
| [[at]] | point  | Pin anchor coordinate and orientation on the sheet border |
| `effects` | node   | Pin text style                                            |
| `uuid`    | uuid   | Unique identifier for this sheet pin                      |

## Interface rule

The parent connects to sheet pins. The child schematic exposes matching `hierarchical_label` elements. The names are the contract between parent and child.
